# Jounetverse

A custom modpack built on [CobbleVerse 1.7.2](https://modrinth.com/modpack/cobbleverse) for a family & friends survival server.

Full Minecraft survival — hunger, mobs, the Ender Dragon — combined with Pokémon adventures powered by Cobblemon.

## What's different from CobbleVerse

- Traveler's Titles — epic RPG-style titles when entering biomes & dimensions
- Tuned for a balanced survival + Pokémon experience

## How to install

### Option A — Prism Launcher (recommended)

1. Download the latest `.mrpack` from the [Releases](../../releases) page
2. Open Prism Launcher → **Add Instance** → **Import**
3. Select the downloaded `.mrpack` file
4. Launch and play

### Option B — Import from URL

1. Open Prism Launcher → **Add Instance** → **Import**
2. Paste the direct download URL of the latest release `.mrpack`
3. Launch and play

## Updating

When a new version is released:
1. Download the new `.mrpack` from [Releases](../../releases)
2. Import it in Prism Launcher as a new instance (or replace the existing one)
